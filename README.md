## 🚀 What is this?
**Fahhh** is a lightweight, cross-browser extension that turns your internet failures into meme-gold. Whenever you encounter a **404 Not Found** or a **Network Error**, Fahhh automatically plays the viral, high-pitched gasp sound effect directly in the background.

No tabs. No overlays. Just pure, unadulterated emotional damage for your browser.

## ✨ Features
- **🌍 Multi-Browser Support**: Native versions for both Firefox and Chrome/Brave/Edge.
- **⚡ Zero Latency**: Sounds trigger instantly when a main frame error is detected.
- **🛡️ Privacy First**: Zero data collection. No tracking. Just gasps.
- **☁️ Invisible Bio-Audio**: 
  - **Firefox**: Runs in a hidden event page.
  - **Chromium**: Utilizes the modern Offscreen API for seamless background playback.

## 🛠️ Installation

### 🦊 Firefox (Official Store)
**The easiest way to installation!** Get the verified version directly from the Firefox Add-ons store:
👉 **[Install Fahhh on Firefox](https://addons.mozilla.org/en-US/firefox/addon/fahhh/)**

No manual setup required. Just click and gasp.

*Manual setup (for developers):* Load `firefox/manifest.json` as a temporary add-on in `about:debugging`.

### 🦁 Chrome / Brave / Edge
**Manual setup required (Chromium):**
1. Download this repo.
2. Go to `chrome://extensions`.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select the `chrome/` folder.
